## Agent skills

### Issue tracker

Issues live in GitHub Issues on `psibrunosg/bstrainer`, via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Default five-label vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context: root `CONTEXT-MAP.md` + per-package `CONTEXT.md` in `apps/web`, `packages/db`, `packages/domain`, `packages/engine`. See `docs/agents/domain.md`.
