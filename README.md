# bstrainer

SaaS de planejamento, execução e análise de treinamento de força. Personal trainers prescrevem pra alunos; usuário solo monta o próprio treino. Três motores de prescrição (templates → manual assistido → IA revisada) escalonados por plano.

Plano completo de arquitetura e features: [docs/ARQUITETURA.md](docs/ARQUITETURA.md).

**Stack:** Next.js (App Router) + Supabase, monorepo pnpm/Turborepo.

**Status:** planejamento concluído, implementação não iniciada.
