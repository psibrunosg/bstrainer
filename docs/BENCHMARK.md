# Benchmark de mercado — apps de treino (pesquisa 13/07/2026)

Apps analisados: Hevy/Hevy Coach, Strong, Fitbod, Boostcamp, Alpha Progression, JuggernautAI, SmartGym, Gravl, Ladder, NTC/NRC, KeyLifts, MFIT Personal, Tecnofit, Queima Diária.

## Top 15 features pro bstrainer (impacto ÷ esforço)

1. **Última performance na tela de log** — consenso absoluto (ghost text peso×reps da última sessão em cada série). Esforço baixo, impacto máximo.
2. **Rest timer automático ao completar série** — default por exercício, override ±15s.
3. **PR detectado + micro-celebração** — badge/haptic no momento do set. Retenção comprovada (NRC: conquista no dia 1 = +64% retenção).
4. **Gráfico e1RM por exercício** — métrica única mais útil (Strong). ✅ já temos.
5. **Check de série gigante + campos numéricos grandes** — dedo suado, olhada de 2s.
6. **Supersets + warm-up/drop/failure sets** — padrão Hevy, esperado por intermediários.
7. **Chat coach-aluno + feed de atividade** (Hevy Coach/MFIT) — coach vê treinos concluídos num feed e comenta. Núcleo da retenção B2B.
8. **Aderência do aluno visível pro coach** (% treinos da semana) — justifica a mensalidade do personal.
9. **Templates de periodização prontos** — Boostcamp provou que biblioteca de programas famosos é aquisição gratuita. ✅ base já existe.
10. **RPE/RIR opcional por série** ✅ já temos.
11. **Plate calculator** — pequeno, adorado, diferencial vs apps BR.
12. **Heatmap de frequência estilo GitHub + streak semanal** (streak por semana, não por dia — não punir descanso).
13. **Clonar rotina e atribuir em escala** (coach) — prescrever pra 20 alunos sem retrabalho.
14. **Cobrança PIX integrada** (MFIT Pay/Tecnofit) — amarra o personal na plataforma. V2.
15. **Heatmap de fadiga muscular** (Fitbod/Gravl) — alto impacto percebido, esforço alto. MVP+1 fim.

## 5 padrões de UX de logger (consenso de mercado)

1. Contexto histórico inline: última carga como ghost text em cada série — nunca sair da tela pra lembrar.
2. Timer auto-disparado ao marcar série, vibração ao fim, ±15s em um toque.
3. Uma tela só durante o treino: lista vertical, séries expandidas, check grande, sem navegação profunda.
4. Dark default + tipografia numérica grande.
5. Feedback de PR na hora do set, não só no resumo final.

## 3 oportunidades que ninguém resolveu

1. **Logger de elite + lado coach completo no mesmo produto.** Hevy Coach: só prescrição/chat. MFIT/Tecnofit: gestão+cobrança mas logger fraco. bstrainer = "Hevy pro aluno + MFIT pro personal" — ninguém no Brasil entrega os dois.
2. **Periodização visível pro aluno.** Mostrar semana/bloco atual (acumulação→intensificação→deload) com barra de progresso do mesociclo. Retenção barata que nenhum mainstream faz.
3. **PWA/web de primeira classe offline-first.** Concorrentes fortes são mobile-native; web deles é fraca. Exatamente a stack do bstrainer.
